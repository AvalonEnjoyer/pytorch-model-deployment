# 🍔 FoodVision Big – Food Image Classification

A deep learning project that classifies food images using a convolutional neural network trained on a large-scale dataset.

This repository contains the notebook and supporting files used to build and run the **FoodVision Big demo**, inspired by modern computer vision workflows.

---

## 🚀 Project Overview

FoodVision Big is designed to:

* Classify food images into multiple categories
* Demonstrate transfer learning and fine-tuning
* Showcase an end-to-end deep learning workflow

The project walks through:

1. Data loading and preprocessing
2. Model building (using transfer learning)
3. Training and evaluation
4. Making predictions on new images

---

## 🧠 Model Details

* Architecture: Transfer learning (e.g. EfficientNet)
* Framework: PyTorch
* Input: Food images
* Output: Predicted food class
